# 🐾 PocketPal

A small desktop companion that lives on your taskbar and walks around your screen.

PocketPal is a lightweight Windows desktop pet built with C# and WPF. It supports animated characters, movement, tray controls, and customisation settings.

## ✨ Features

- 🐶 Animated desktop companion
- 🚶 Walks around your screen
- 🖱️ Click the taskbar area to make your pet run to that location
- 🎞️ Sprite-based animations
- 🔄 Idle and movement states
- 📌 Stays visible above the taskbar
- 🖱️ Optional click-through behaviour
- ⚙️ Settings menu from the system tray
- 🔌 Runs in the background
- 💾 Saves user settings

## 🛠️ Built With

- C#
- .NET
- WPF
- Windows Forms NotifyIcon
- Custom animation/state system
